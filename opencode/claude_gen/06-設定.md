# 06. 設定（opencode.json）

> **概要**: `opencode.json` の場所・優先順位・主要キー・変数置換をまとめる。
> **参照元**: <https://opencode.ai/docs/ja/config>

opencode は **JSON** と **JSONC**（コメント付き JSON）をサポートする。スキーマは `https://opencode.ai/config.json`。

---

## 設定ファイルの場所と優先順位（後のものが優先）

1. **リモート**（`.well-known/opencode` エンドポイント）— 組織デフォルト
2. **グローバル**（`~/.config/opencode/opencode.json`）— ユーザー全体
3. **カスタム**（`OPENCODE_CONFIG` 環境変数のパス）
4. **プロジェクト**（プロジェクトルートの `opencode.json`）
5. **`.opencode` ディレクトリ**（エージェント・コマンド・プラグイン）
6. **インライン**（`OPENCODE_CONFIG_CONTENT` 環境変数）— 実行時上書き

> 設定は**マージ**される（置換ではない）。競合キーは上書きされ、競合しない設定は全ソースから保持される。

TUI 固有の設定は別ファイル `tui.json`（[05-TUIとキーバインド.md](05-TUIとキーバインド.md)）。

---

## 主要な設定キー

```json
{
  "$schema": "https://opencode.ai/config.json",
  "model": "anthropic/claude-sonnet-4-5",
  "small_model": "anthropic/claude-haiku-4-5",
  "provider": {
    "anthropic": {
      "options": { "timeout": 600000, "setCacheKey": true }
    }
  },
  "autoupdate": true,
  "theme": "opencode",
  "default_agent": "plan",
  "tools": { "write": false, "bash": false },
  "permission": { "edit": "ask", "bash": "ask" },
  "agent": {
    "code-reviewer": {
      "description": "Reviews code",
      "model": "anthropic/claude-sonnet-4-5",
      "prompt": "You are a code reviewer...",
      "tools": { "write": false }
    }
  },
  "command": {
    "test": { "template": "Run tests with coverage", "description": "Run tests", "agent": "build" }
  },
  "instructions": ["CONTRIBUTING.md", "docs/guidelines.md"],
  "mcp": {},
  "share": "manual",
  "server": { "port": 4096, "hostname": "0.0.0.0", "mdns": true },
  "formatter": { "prettier": { "disabled": true } },
  "watcher": { "ignore": ["node_modules/**", "dist/**"] },
  "compaction": { "auto": true, "prune": false },
  "enabled_providers": ["anthropic", "openai"],
  "disabled_providers": ["gemini"],
  "plugin": ["opencode-helicone-session"]
}
```

主なキーの概要:

| キー | 説明 |
| --- | --- |
| `model` / `small_model` | 既定モデル / 軽量タスク用モデル |
| `provider` | プロバイダー別オプション（API キー・タイムアウト等） |
| `autoupdate` | 自動更新の有効/無効 |
| `theme` | TUI テーマ |
| `default_agent` | 起動時のエージェント |
| `tools` | ツールの有効/無効（`permission` に統合済みだが後方互換で残存） |
| `permission` | 操作の承認方式（[10-権限とツール.md](10-権限とツール.md)） |
| `agent` | カスタムエージェント（[08-エージェント.md](08-エージェント.md)） |
| `command` | 再利用コマンド（[09-コマンド.md](09-コマンド.md)） |
| `instructions` | 追加指示ファイル（[07-ルール-AGENTSmd.md](07-ルール-AGENTSmd.md)） |
| `mcp` | MCP サーバー（[11-MCP.md](11-MCP.md)） |
| `formatter` | コードフォーマッタ |
| `enabled_providers` / `disabled_providers` | プロバイダーの許可 / 除外 |

---

## 変数置換

### 環境変数（`{env:...}`）

```json
{
  "model": "{env:OPENCODE_MODEL}",
  "provider": {
    "anthropic": { "options": { "apiKey": "{env:ANTHROPIC_API_KEY}" } }
  }
}
```

### ファイル内容（`{file:...}`）

```json
{
  "instructions": ["./custom-instructions.md"],
  "provider": {
    "openai": { "options": { "apiKey": "{file:~/.secrets/openai-key}" } }
  }
}
```

ファイルパスは設定ファイルからの相対、または絶対（`/` か `~` 始まり）。
