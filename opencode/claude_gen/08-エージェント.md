# 08. エージェント

> **概要**: 組み込みエージェント（Build / Plan など）と、カスタムエージェントの作成をまとめる。
> **参照元**: <https://opencode.ai/docs/ja/agents>

opencode のエージェントには、直接対話する **プライマリエージェント**と、特定タスクを担う **サブエージェント**がある。

---

## 組み込みエージェント

### プライマリ（`Tab` で切替）

| エージェント | 説明 |
| --- | --- |
| **Build** | 既定。全ツールアクセスを持つ開発用エージェント |
| **Plan** | 分析・計画向けの制限付きエージェント（実際の変更はしない） |
| Compact / Title / Summary | 自動実行される隠しシステムエージェント |

### サブエージェント（自動または `@メンション`で呼び出し）

| エージェント | 説明 |
| --- | --- |
| **General** | 全ツールアクセス。複雑なマルチステップタスク |
| **Explore** | 読み取り専用のコードベース探索 |
| **Scout** | 外部ドキュメント・依存関係の読み取り専用調査 |

- プライマリは `Tab` キーで切り替える。
- サブエージェントはプライマリから自動的に、またはメッセージ内の `@agent-name` で手動呼び出しできる。

---

## カスタムエージェントの作成

3 つの方法がある:

1. **対話 CLI**: `opencode agent create`
2. **Markdown ファイル**: `.opencode/agents/`（プロジェクト）または `~/.config/opencode/agents/`（グローバル）
3. **JSON 設定**: `opencode.json` の `agent`

### フロントマター / 設定フィールド

| フィールド | 説明 |
| --- | --- |
| `description`（必須） | エージェントの役割 |
| `mode` | `primary` / `subagent` / `all` |
| `model` | 使用するモデル |
| `temperature` | 応答のランダム性（0.0〜1.0） |
| `prompt` | カスタムシステムプロンプト（ファイル） |
| `tools` | ツールの有効/無効 |
| `permission` | 操作制御（`ask` / `allow` / `deny`） |

### 例（Markdown）

```markdown
---
description: Reviews code for quality and best practices
mode: subagent
model: anthropic/claude-sonnet-4-20250514
temperature: 0.1
tools:
  write: false
  edit: false
  bash: false
---
You are a code reviewer. Focus on security, performance,
and maintainability.
```

### 例（`opencode.json`）

```json
{
  "agent": {
    "code-reviewer": {
      "description": "Reviews code",
      "mode": "subagent",
      "model": "anthropic/claude-sonnet-4-5",
      "prompt": "You are a code reviewer...",
      "tools": { "write": false }
    }
  }
}
```
