# 01. インストール（Linux / CLI）

> **概要**: opencode を Linux にインストール・更新・アンインストールする手順をまとめる。
> **参照元**: <https://opencode.ai/docs/ja/>

opencode はオープンソースの AI コーディングエージェント。ターミナル（TUI）で動作し、コードベースの質問応答・機能追加・コード変更を支援する。

---

## インストール方法

### 方法1: インストールスクリプト（推奨）

```bash
curl -fsSL https://opencode.ai/install | bash
```

### 方法2: Node パッケージマネージャ

```bash
npm install -g opencode-ai
bun install -g opencode-ai
pnpm install -g opencode-ai
yarn global add opencode-ai
```

### 方法3: Homebrew

```bash
brew install anomalyco/tap/opencode
```

### 方法4: Arch Linux

```bash
sudo pacman -S opencode      # 安定版（公式リポジトリ）
paru -S opencode-bin         # 最新版（AUR）
```

---

## 起動

プロジェクトディレクトリで `opencode` を実行:

```bash
cd /path/to/project
opencode
```

初回は認証（プロバイダー / API キー）の設定が必要（[02-認証とプロバイダー.md](02-認証とプロバイダー.md)）。

---

## 更新

```bash
opencode upgrade              # 最新版へ
opencode upgrade v0.1.48      # 特定バージョン
opencode upgrade --method npm # インストール方法を指定（curl/npm/pnpm/bun/brew）
```

自動更新を無効化する場合は環境変数 `OPENCODE_DISABLE_AUTOUPDATE=1`、または設定で `"autoupdate": false`（[06-設定.md](06-設定.md)）。

---

## アンインストール

```bash
opencode uninstall
# オプション
#   --keep-config, -c   設定を残す
#   --keep-data, -d     セッション・スナップショットを残す
#   --dry-run           削除内容をプレビュー
#   --force, -f         確認をスキップ
```

手動で削除する場合の主なパス（Linux）:
- 設定: `~/.config/opencode/`
- データ（セッション・認証情報・ログ）: `~/.local/share/opencode/`
- キャッシュ: `~/.cache/opencode/`

---

## Linux 固有の注意（クリップボード）

TUI のコピー&ペーストにはクリップボードユーティリティが必要:

```bash
# X11
sudo apt install xclip     # または xsel
# Wayland
sudo apt install wl-clipboard
# ヘッドレス
sudo apt install xvfb
```

Wayland で問題が出る場合は `OC_ALLOW_WAYLAND=1 opencode` を試すか、X11 セッションに切り替える。
