# 03. クイックスタート

> **概要**: インストール・認証後、opencode を使い始める流れをまとめる。
> **参照元**: <https://opencode.ai/docs/ja/> / <https://opencode.ai/docs/ja/tui>

---

## 手順

### 1. インストール（詳細は [01-インストール.md](01-インストール.md)）

```bash
curl -fsSL https://opencode.ai/install | bash
```

### 2. プロジェクトで起動

```bash
cd /path/to/project
opencode
```

### 3. 認証（詳細は [02-認証とプロバイダー.md](02-認証とプロバイダー.md)）

TUI 内で:

```text
/connect
```

プロバイダーを選び、API キーを設定する。

### 4. AGENTS.md を生成（`/init`）

```text
/init
```

コードベースを分析して、プロジェクトルートに `AGENTS.md`（プロジェクトの永続指示ファイル）を生成する。git にコミットしてチーム共有できる（[07-ルール-AGENTSmd.md](07-ルール-AGENTSmd.md)）。

### 5. 会話する

自然言語で依頼する。主な入力方法:
- `@` でファイル参照（作業ディレクトリ内をファジー検索）。
- `!` で始めるとシェルコマンドを直接実行し、出力を会話に取り込む。

```text
このプロジェクトの構成を説明して
@src/auth.ts のロジックを説明して
! npm test
ユーザー登録フォームに入力検証を追加して
```

### 6. エージェント / モデルを切り替える

- `Tab` でエージェント（Build / Plan など）を循環（[08-エージェント.md](08-エージェント.md)）。
- `/models`（`ctrl+x m`）でモデル切替。

---

## 非対話実行（`opencode run`）

TUI を開かずにプロンプトを実行:

```bash
opencode run "Explain how closures work in JavaScript"
git log --oneline -20 | opencode run "要約して"
opencode run --format json "..."   # JSON 出力
```

（詳細は [04-CLIリファレンス.md](04-CLIリファレンス.md)）

---

## 覚えておくと便利な操作

| 操作 | 方法 |
| --- | --- |
| 新しいセッション | `/new`（`ctrl+x n`） |
| セッション一覧・切替 | `/sessions`（`ctrl+x l`） |
| コンテキスト圧縮 | `/compact`（`ctrl+x c`） |
| 直前を元に戻す / やり直す | `/undo`（`ctrl+x u`）/ `/redo`（`ctrl+x r`） |
| 外部エディタで入力 | `/editor`（`ctrl+x e`） |
| ヘルプ | `/help`（`ctrl+x h`） |
| 終了 | `/exit`（`ctrl+x q`） |

`ctrl+x` は **リーダーキー**（[05-TUIとキーバインド.md](05-TUIとキーバインド.md)）。
